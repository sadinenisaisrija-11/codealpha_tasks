# 🛒 Simple E-Commerce Store

## 📖 Description
A simple E-Commerce Store website developed as part of the CodeAlpha Full Stack Development Internship. The website allows users to browse products, view product details, add products to a shopping cart, login, and place orders.

## 🚀 Features
- User Login Page
- Product Listing Page
- Product Details Page
- Shopping Cart Functionality
- Add to Cart Feature
- Cart Item Counter
- Place Order Functionality
- Responsive User Interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## ▶️ How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Explore the E-Commerce Store features.

## 📋 Functionalities
### Login
- User enters username and password.
- Redirects to the Home Page after successful login.

### Shopping Cart
- Add products to cart.
- Cart count updates automatically.
- Selected products are displayed in the cart section.

### Order Processing
- Users can place an order using the "Place Order" button.
- Confirmation message is displayed after order placement.
